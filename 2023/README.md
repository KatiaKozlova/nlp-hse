## [ДЗ1](АвтОбрЕЯ_Козлова_ДЗ1.ipynb). Разрешение семантической многозначности (WSD).
Кластеризация контекстов для неоднозначных слов и анализ, как результаты соотносятся с их значениями.
1. Выбор 5 неоднозначных лексем и сбор их словарных значений (толкований) при помощи краулинга.
2. Выбор нескольких корпусов из [коллекции Corus](https://github.com/natasha/corus#reference), извлечение оттуда всех предложений, где содержатся данные неоднозначные слова.
3. Разбиение полученных контекстов по значениям двумя любыми способами.
4. Приписывание им значения автоматически.
5. Выбор 2-3 значений для слова и по 5 контекстов, которые автоматически были отнесены к этим значениям. Разметка контекстов: совпадает ли значение слова в контексте со словарным значением.
6. Оценка accuracy — в какой доле контекстов автоматически определенно значение для контекста совпадает с автоматически определенным значением толкования.
7. Сравнение результатов двух методов WSD и объяснение, почему результаты совпадают или расходятся.

## [ДЗ2](АвтОбрЕЯ_Козлова_ДЗ2/АвтОбрЕЯ_Козлова_ДЗ2.ipynb). Сравнение систем автоматического распознавания речи.
Сравнение автоматических субтитров YouTube с другими системами.
1. Выбор видео на YouTube, где есть речь и автоматические субтитры и скачивание звука видео и субтитров (файлы также находятся в [репозитории](АвтОбрЕЯ_Козлова_ДЗ2)).
2. Выбор 3-х систем ASR и применение их к файлу со звуком.
3. Создание эталона и оценка качества работ систем и автогенерации YouTube при помощи WER для каждой системы.
